# 肌肉协同关系与训练编排逻辑

本文件基于 ExerciseDB 数据集（1,324个动作）分析得出，揭示动作之间的肌肉连锁关系，是排训练计划的核心参考。

---

## 一、核心协同关系总览

每个动作都有三层肌肉参与：
- **主练肌（target）**：该动作主要刺激的肌群
- **协同肌（muscle_group）**：第二大参与肌群
- **辅助肌（secondary_muscles）**：间接参与的其他肌群

### 关键数据发现

| 肌群 | 作为主练的动作数 | 作为辅助被消耗的次数 | 意义 |
|------|-----------------|---------------------|------|
| 三头肌 | 141 | **268次** | 几乎所有推类动作都会消耗三头 |
| 前臂 | 37 | **277次** | 几乎所有拉类动作都消耗前臂（握力） |
| 二头肌 | 151 | **194次** | 几乎所有拉类动作都会消耗二头 |
| 腘绳肌 | 28 | **289次** | 几乎所有下肢动作都涉及腘绳肌 |
| 臀大肌 | 144 | **136次** | 下肢动作的核心参与者 |
| 下背 | — | **71次** | 硬拉、深蹲、甚至腹肌动作都会消耗下背 |

## 二、推/拉/腿三分化的协同逻辑

### 推日（胸+肩+三头）的连锁反应

```
杠铃卧推 → 主练:胸 | 协同消耗:三头 | 辅助消耗:三角肌前束
上斜哑铃卧推 → 主练:胸 | 协同消耗:三角肌前束 | 辅助消耗:三头
双杠臂屈伸 → 主练:胸 | 协同消耗:三头 | 辅助消耗:三角肌前束
推举 → 主练:三角肌 | 协同消耗:三头 | 辅助消耗:上背
Y字侧平举 → 主练:三角肌 | 协同消耗:斜方肌 | 辅助消耗:上背
```

**推日结论：** 做完卧推+上斜+臂屈伸后，三头肌已经消耗了70%以上。此时再加三头孤立动作（如仰卧屈伸），训练量要控制在2-3组即可，不需要像单独练三头那样做5-6组。

### 拉日（背+后束+二头）的连锁反应

```
钢线下拉 → 主练:背阔肌 | 协同消耗:二头 | 辅助消耗:菱形肌、后束
器械划船 → 主练:上背 | 协同消耗:二头 | 辅助消耗:前臂
对握下拉 → 主练:背部中段 | 协同消耗:二头
开肘下拉 → 主练:后束+上背 | 协同消耗:二头
钢线弯举 → 主练:二头 | 协同消耗:前臂
```

**拉日结论：** 5个背部动作做完后，二头肌已经被大量间接消耗。弯举只需2-3组收尾即可。如果二头是弱项想重点练，可以把弯举放在背部动作之前做（预疲劳法），但要接受背部训练效果会因此打折。

### 腿日（股四+腘绳）的连锁反应

```
高脚杯深蹲 → 主练:股四 | 协同消耗:臀大肌 | 辅助消耗:腘绳肌、小腿
保加利亚蹲 → 主练:股四 | 协同消耗:臀大肌 | 辅助消耗:腘绳肌
罗马尼亚硬拉 → 主练:臀/腘绳 | 协同消耗:腘绳肌 | 辅助消耗:下背
单腿硬拉 → 主练:臀 | 协同消耗:腘绳肌 | 辅助消耗:下背
山羊挺身 → 主练:竖脊肌 | 协同消耗:臀 | 辅助消耗:腘绳肌
```

**腿日结论：** 整条后链（腘绳肌→臀→下背）在腿日被全面消耗。这意味着腿日后的第二天不应该安排拉日（硬拉/划船），因为下背和腘绳肌还没恢复。

## 三、分化安排的底层逻辑

### 为什么推日和拉日之间最好隔一天？

- 推日大量消耗三头 → 三头需要48-72小时恢复
- 拉日大量消耗二头 → 二头需要48-72小时恢复
- 虽然三头和二头是不同肌群，但前臂/握力在两天都会被消耗
- 如果连续练推→拉，握力疲劳会影响拉日动作质量

### 为什么腿日要单独放一天？

- 腿部肌群体积最大（股四+腘绳+臀），训练后全身疲劳和代谢压力最高
- 下背在拉日已经承受负荷（划船、硬拉变体），腿日的深蹲/硬拉再次消耗下背
- 腘绳肌在拉日的罗马尼亚硬拉中已被激活，腿日需要它做更多功

### 最优分化排列

```
推荐1: 推 → 腿 → 休 → 拉 → 休 → 推 → 腿 → 拉（8天循环）
推荐2: 推 → 拉 → 休 → 腿 → 休（5天循环，适合每周3-4次）
避免: 腿 → 拉（下背和腘绳未恢复）
避免: 推 → 推（三头和前束未恢复）
```

## 四、动作编排顺序原则

### 同一训练日内的动作排序

1. **复合动作优先**：多关节、大肌群动作放前面（卧推、深蹲、硬拉）
2. **孤立动作在后**：单关节动作放后面（飞鸟、弯举、下压）
3. **弱项优先**：如果某个肌群是弱项，在状态最好时先练它
4. **预疲劳法（高级）**：先做孤立动作再做复合动作，适合有经验的训练者突破瓶颈

### 需要避免的动作组合

| 避免组合 | 原因 |
|----------|------|
| 卧推后立即做大量三头下压 | 三头已在卧推中消耗，再做效果差且易过度训练 |
| 划船后立即做杠铃弯举 | 二头已在划船中消耗，弯举只需少量即可 |
| 硬拉次日安排大重量划船 | 下背未恢复，极易受伤 |
| 深蹲和硬拉放同一天 | 下背承受双重负荷，风险极高（除非有充分训练基础） |
| 肩部有伤者做推举+侧平举+卧推 | 三角肌前束过度消耗，加重肩部负担 |

### 推荐的协同搭配

| 搭配 | 效果 |
|------|------|
| 卧推 + 蝴蝶机夹胸 | 先复合后孤立，胸肌全面刺激 |
| 下拉 + 划船 | 不同角度刺激背部，协同消耗二头 |
| 深蹲 + 罗马尼亚硬拉 | 股四+腘绳前后兼顾 |
| 面拉 + 反向飞鸟 | 后束双重刺激，同时强化肩胛稳定 |
| 背部训练 + 少量二头弯举 | 顺应协同消耗，2-3组即可到位 |

## 五、训练量分配参考

基于协同消耗的训练量建议（每组按RPE8计算）：

### 推日训练量分配

| 肌群 | 总组数建议 | 分配方式 |
|------|-----------|----------|
| 胸 | 10-14组 | 复合动作8-10组 + 孤立2-4组 |
| 三角肌前束 | 已被卧推覆盖 | 不需要额外训练 |
| 三角肌中束 | 3-4组 | 侧平举类动作 |
| 三角肌后束 | 0-2组 | 推日不是重点 |
| 三头 | 4-6组 | 已在推类动作中消耗70%，补充即可 |

### 拉日训练量分配

| 肌群 | 总组数建议 | 分配方式 |
|------|-----------|----------|
| 背 | 12-16组 | 下拉4-6组 + 划船4-6组 + 其他2-4组 |
| 后束 | 3-4组 | 面拉/反向飞鸟 |
| 二头 | 3-4组 | 已在拉类动作中消耗，收尾即可 |
| 前臂 | 不需要额外训练 | 握力在拉类动作中已被充分消耗 |

### 腿日训练量分配

| 肌群 | 总组数建议 | 分配方式 |
|------|-----------|----------|
| 股四 | 8-12组 | 深蹲类6-8组 + 孤立2-4组 |
| 臀/腘绳 | 8-10组 | 硬拉类6-8组 + 孤立2组 |
| 下背 | 3-4组 | 山羊挺身即可，不要额外加 |
| 小腿 | 可选3-4组 | 如需要可加站姿提踵 |
