# 训练编排决策树

本文件是排训练计划时的完整决策流程。AI在为用户编排计划时，应按此流程逐步执行。

---

## 第一步：用户评估

收集以下信息：
1. 训练目标：增肌 / 减脂 / 体态改善 / 提升体能 / 力量提升
2. 训练经验：零基础 / <半年 / 半年-1年 / 1-2年 / 2年以上
3. 伤病情况：肩 / 腰 / 膝 / 腕 / 无
4. 可用器材：健身房全套 / 家庭哑铃 / 纯自重
5. 每周可训练天数：2天 / 3天 / 4天 / 5天+
6. 弱项肌群（可选）

---

## 第二步：选择分化模式

根据经验和天数决定：

```
零基础 + 任意天数 → 一分化（全身），2-4周后升级
<半年 + 2-3天/周 → 二分化（上下肢 或 推+拉腿）
<半年 + 4天/周 → 二分化（躯干+四肢），4周后升级
半年-1年 + 3天/周 → 三分化（推拉腿）
半年-1年 + 4天/周 → 三分化+有氧日
1年以上 + 3-4天/周 → 三分化（凯圣王体系）
1年以上 + 5天/周 → 三分化 + 弱项补强日
```

特殊目标调整：
- 减脂为主 → 任何分化 + 碳循环饮食方案
- 力量为主 → 降低次数(3-5次)，增加组间休息(2-3分钟)
- 体态改善 → 加入体态纠正模块（三阶段热身必做）

---

## 第三步：确定训练日内容

### 3.1 动作选择原则

1. 复合动作优先（多关节、大肌群）
2. 每个训练日选5-7个动作
3. 动作覆盖所有目标肌群
4. 考虑肌肉协同关系（参考muscle-synergy.md）

### 3.2 推日模板

```
热身 → 三阶段热身（重点肩胛激活）
动作1：胸部复合动作（卧推类）→ 3-4组
动作2：胸部上斜动作 → 3-4组
动作3：胸部孤立动作（飞鸟类）→ 3组
动作4：肩部推举或侧平举 → 3组（肩伤则跳过推举）
动作5：后束/侧平举 → 3组
动作6：三头孤立（可选）→ 2-3组（三头已在推类中消耗70%）
收尾：核心训练
```

### 3.3 拉日模板

```
热身 → 背部激活（弹力带肩胛后缩+直臂下拉）
动作1：垂直拉（下拉/引体类）→ 4组
动作2：水平拉（划船类）→ 4组
动作3：垂直拉变体 → 3组
动作4：后束孤立（面拉/反向飞鸟）→ 3组
动作5：二头孤立（可选）→ 2-3组（二头已在拉类中消耗）
收尾：核心训练
```

### 3.4 腿日模板

```
热身 → 下肢专项热身（髋/膝/踝活动度）
动作1：股四主导（深蹲类）→ 4组
动作2：单腿动作（保加利亚蹲/单腿硬拉）→ 3组/侧
动作3：臀/腘绳主导（硬拉类）→ 4组
动作4：后链孤立（山羊挺身/腿弯举）→ 3组
动作5：小腿（可选）→ 3-4组
收尾：核心训练
```

---

## 第四步：协同校验

排完计划后，检查以下项目：

### 4.1 日内协同检查
- [ ] 推类动作后，三头总组数是否超过8组？（超过则减量）
- [ ] 拉类动作后，二头总组数是否超过6组？（超过则减量）
- [ ] 是否有两个动作主练同一肌群且都是高疲劳动作？（应一个复合+一个孤立）
- [ ] 肩部有伤者是否避免了过顶推举？

### 4.2 日间协同检查
- [ ] 腿日后是否安排了拉日？（应避免，下背/腘绳未恢复）
- [ ] 推日后是否安排了推日？（应避免，三头/前束未恢复）
- [ ] 同一肌群两次训练间隔是否≥48小时？

### 4.3 训练量检查
- [ ] 每个肌群每周总组数是否在10-20组之间？（低于10刺激不足，高于20恢复困难）
- [ ] 单次训练总组数是否在15-25组之间？（超过25则训练过长）
- [ ] 新手单次训练是否控制在45-60分钟？

---

## 第五步：渐进超负荷策略

### 新手（<1年）
- 每次训练尝试增加1-2次重复次数
- 达到目标次数上限后，增加重量5-10%
- 加重后次数可能下降，这是正常的

### 中级（1-2年）
- 采用"双渐进法"：先增次数到上限，再加重量
- 每4-6周安排一个减载周（降低50%训练量）
- 可引入高级技术：超级组、递减组、暂停组

### 高级（2年+）
- 周期化训练：力量周期→容量周期→减载周期
- 力量周期：3-5次/组，4-5组
- 容量周期：8-15次/组，3-4组
- 每8-12周一个完整周期

---

## 第六步：伤病适配

如果有伤病，参考 injury-substitutions.md 进行动作替代：
- 肩伤：跳过所有过顶动作，用地板卧推代替平板卧推，加强后束和肩胛稳定
- 腰伤：用器械腿举代替深蹲/硬拉，加入核心稳定训练
- 膝伤：减少深蹲幅度，用髋主导动作替代膝主导动作
- 腕伤：使用中立握法、护腕、或器械替代杠铃动作

---

